# Contributing to Chain of Lands

Thank you for your interest in contributing to Chain of Lands.

This project is a long-term world-building initiative that values
consistency, clarity, and deliberate progress.
All contributions are expected to respect these principles.

---

## 1. Contribution Philosophy

Chain of Lands is not built for speed.
It is built for endurance.

Contributions are evaluated based on:
- Alignment with the project vision
- Respect for canonical lore
- Long-term maintainability
- Clarity of intent

Not all ideas will be accepted,
but all constructive contributions are welcome.

---

## 2. Types of Contributions

We currently welcome the following forms of contribution:

### Technical Contributions
- Frontend improvements
- Interactive map enhancements
- Tooling and infrastructure support
- Documentation improvements

### Narrative Contributions
- Lore expansions (non-canonical proposals)
- Faction concepts
- World-building suggestions
- Narrative feedback

### Documentation
- Typos and clarity improvements
- Structural refinements
- Explanatory additions

---

## 3. What We Do Not Accept

To preserve consistency, the following are not accepted at this stage:

- Direct changes to canonical lore
- Speculative financial content
- Marketing or promotional material
- Token price discussion or guarantees

Canonical changes require explicit approval.

---

## 4. Contribution Process

Please follow this process for all contributions:

1. Fork the repository
2. Create a new branch from `main`
3. Make your changes with clear intent
4. Ensure consistency with existing documents
5. Submit a pull request with a detailed description

Pull requests without context may be closed without review.

## 6. Review Process

All contributions are reviewed manually.

Reviews may include:
- Technical evaluation
- Narrative consistency checks
- Architectural alignment

Feedback is part of the process.
Revisions may be requested.

---

## 7. Communication

For questions or proposals:
- Use GitHub Issues
- Clearly describe intent and scope
- Avoid assumptions about acceptance

Respectful and constructive communication is required at all times.

---

## 8. Code of Conduct

All contributors are expected to:
- Act respectfully
- Avoid harassment or hostility
- Engage in good faith discussions

Chain of Lands is a collaborative space.

---

## Closing Note

Contributing to Chain of Lands means contributing to a world,
not just a codebase.

If you value patience, consistency, and long-term thinking,
you are welcome here.
